## Setup

- Fork & Clone
- Run `bundle install` to install dependencies.
- Run `bin/rake db:reset` to create, load and seed db.
- Run `bin/rails s -b 0.0.0.0` to start the server.

## Dependencies

* Rails 4.2 
* PostgreSQL 9.x
* Stripe
